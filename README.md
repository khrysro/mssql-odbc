# ppoffice/mssql-odbc
Dockerfile of Ubuntu Trusty with Microsoft SQL Server ODBC Driver Support

```bash
docker pull ppoffice/mssql-odbc
```

## Build
```bash
docker build -t ppoffice/mssql-odbc .
```

## Running
```bash
docker run -it ppoffice/mssql-odbc:latest /bin/bash
```
