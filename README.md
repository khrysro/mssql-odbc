# ppoffice/apache-php-odbc-mssql
Dockerfile of Ubuntu Trusty with Microsoft SQL Server ODBC Driver Support

## Build
```bash
docker build -t ppoffice/microsoft-odbc .
```

## Running
```bash
docker run -it ppoffice/microsoft-odbc /bin/bash
```
